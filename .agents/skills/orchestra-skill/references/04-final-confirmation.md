# Workflow: Final Confirmation & Hand-off

The final step to wrap up the task.

- **Goal:** To report the results, receive feedback, and conclude the task.
- **Input:** The completed and tested code.
- **Process:**
    1.  **Summarize Changes:** List the files that were created or modified.
    2.  **Summarize Work:** Present a summary of what was done and the results achieved.
    3.  **Suggest Next Steps:** Propose subsequent actions (e.g., "Do you want to deploy to Vercel?", "Should I create a Pull Request?").
    4.  **Request Confirmation:** Ask the user if the task has been completed to their satisfaction.
- **Output:** User confirmation and the end of the session.
